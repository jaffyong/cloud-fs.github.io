V0.7.16(Web1.5.14.0)

1. 移动文件策略修改：通过挂载盘和WebDAV移动文件或文件夹时，和本地文件系统的行为保持一致，即如果要移动的目标文件已存在，会直接替换目标文件，如果要移动的目标文件夹已存在且不为空时，移动会失败；通过Web界面移动文件文件或文件夹时，如果目标文件或文件夹已存在，会提示选择移动策略，可以选择忽略、替换、重命名或者取消，注意：如果同名文件夹选择覆盖，会直接删除目标文件夹，不会深度分析文件夹下的文件覆盖；同时移动多个文件或文件夹时，没有冲突的文件或文件夹正常移动不受策略影响；通过web界面跨盘移动（复制）文件或文件夹改回原来的策略，即同名文件跳过
2. Linux和Android版本的本地挂载默认改为特权模式(privileged mode)fuse挂载，特权模式要求运行clouddrive的是root用户，不再要求系统安装fuse3和fusermount3，只要内核支持fuse即可；如果以非root用户运行，会自动切换到以前版本采用的非特权模式，像以前版本一样，非特权模式运行需要系统安装fuse3和fusermount3
3. 修复115网盘和百度网盘下载文件时有可能出现下载失败超时的问题
