V0.7.8(Web1.5.5.0)
1. 增加目录缓存本地持久化功能，该功能是Pro功能，所有付费会员包括轻享终身会员可以使用，在系统设置中打开缓存目录本地持久化开关即可
2. 因PikPak官方调整，暂时移除PikPak支持，需要使用PikPak网盘的用户，请在官方页面打开WebDAV功能，用CloudDrive2的WebDAV连接使用
3. 新增支持在运行clouddrive前设置环境变量LOCAL_ROOT_PATH，设置该环境变量后，添加本地存储时，根目录将被设定为该环境变量设置的值，对本地文件安全性有要求的用户可以设置该环境变量
4. 添加本地目录时支持直接添加根目录
5. 修复部分设备迅雷云盘无法添加的问题，因迅雷云盘策略调整，目前一个迅雷云盘账号只支持一个设备登录，超过一个设备登录同账号迅雷云盘可能会导致某些终端无法使用
6. 修改日志文件的文件名格式，增加后缀.log
7. 其它性能改进和bug修复
