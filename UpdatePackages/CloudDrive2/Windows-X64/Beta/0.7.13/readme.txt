V0.7.13(Web1.5.10.0)
1. 根据115的新规则，修复频繁访问115网盘会被风控的问题，115的MaxQueriesPerSecond默认值和最大值都将被自动限定为4。注意，请不要同时在多个同账户的CD2终端上频繁访问同一个115网盘，否则还有可能被风控
2. 因PikPak官方调整，暂时移除PikPak支持，需要使用PikPak网盘的用户，请在官方页面打开WebDAV功能，用CloudDrive2的WebDAV连接使用
3. 修复在线更新各种不稳定的问题，当前的版本如果无法在线更新，请先手动安装新版本，后续版本更新会更加稳定
4. 修复打开目录缓存本地持久化之后的一些不稳定和死锁的bug
5. 修复直接登录替换现有网盘后，当前正在缓存内的文件没有更新的问题
6. 修复直接登录替换现有网盘后，网盘下载参数及代理设置被重置的问题
7. 其它bug修复、性能改进和界面微调
