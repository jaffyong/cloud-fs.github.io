V0.7.0-beta2(Web1.5.0.2)

1. 增加显示同账号下所有在线的clouddrive2终端信息，包括设备名称，操作系统，版本号和IP地址（仅支持0.7.0-beta2以上版本的终端），设备名称默认为系统主机名，可以在系统设置中修改设备名称
2. 增加踢出同账号下任意在线终端的功能，终端被踢出后将处于彻底退出状态，需要重新输入账号密码才能登录（仅支持踢出0.7.0-beta2以上版本的终端）
3.  增加用户数据实时自动同步：clouddrive2终端，只要登录时打开了账号信息同步到云端，任意登录终端的云盘登录账号信息修改，都会实时自动同步到所有其它打开账号信息同步的在线终端（账号信息同步只能在0.7.0-beta2以上版本的clouddrive2终端之间进行，老版本更新的同步和被同步都不支持）
4. 增加修改用户密码功能，在系统设置中可以修改用户密码。注意：在修改密码前，请把同账号下所有终端都升级到0.7.0-beta2，否则可能造成用户同步到云端的数据丢失。修改密码后所有在线终端都将强制下线，需要重新登录
5. 增加启动后自动运行外部程序设置：在配置目录下systemsettings.json里手动添加"run_after_start"项，可用于在系统启动成功后自动执行一次指定的本机外部程序，修改完配置后重启服务生效。如：
"run_after_start": "service nfsd restart"
出于安全考虑，本选项只提供本地编辑配置文件实现，不支持通过web设置
6. 修复pikpak无法登录的问题
